## Developement

Requirements :
- Node.js
- NPM
 
Run the project locally :
```
npm install
npm run dev
```

Recording conditions:
- Safari (for the best performance)
- [Black Hole](https://existential.audio/blackhole/) to record audio out
- Quicktime screen record
